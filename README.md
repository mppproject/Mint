Mint
====

a small interpreter for MPP

How to use ?
====

You can download the debian package from www.mpp-project.eu , or compiling the sources .

    $ make
    $ ./mint -h
    
    
Mint don't need any header file , so you don't have to add a #include .


Example file
====  

    #bda
    ra9m bda(){
    ktab("test \n");
    }
